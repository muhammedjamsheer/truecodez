JavaScript Variables

Variables are containers for storing data (storing data values).

we can declare javascript variables in 3 types var, let ,const

Var

It is the oldest keyword to declare a variable in js

Variables declared with var are in the function scope and can be Redeclared.

Let

The let keyword was introduced in ES6 (2015)

Variables declared with let have Block Scope ie, Variables declared inside a curly brace cannot be accessed from outside the block.

Variables declared with let cannot be Redeclared and must be declared before use.

Const

The let keyword was introduced in ES6 (2015)

Variables declared with const have Block Scope.
