Interview Questions

1.What is Angular 8

Angular 8 is a client-side TypeScript based framework which is used to create dynamic web applications.
Its first version was released by Google in 2012 and named as AngularJS.
Typescript is a super-set of JavaScript.

Features of Angular8
  • Angular 8 supports TypeScript 3.4
  • Angular 8 provides dynamic imports for lazy-loaded modules.

2.Difference between Angular Js and Angular

Angular JS Angular
Angular JS Does not supported by mobile browsers. Angular supported by all the popular mobile browsers.
AngularJS works on MVC, Model View Controller Design. Angular uses components and directives. Here component is directive with a template.
AngularJS code is written in javascript. Angular code is written in typescript.

1.What is a @ViewChild ?

A ViewChild is a component if we want to access a child component, directive, DOM element inside the parent component, we use the decorator @ViewChild() in Angular.
