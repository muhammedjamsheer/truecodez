ng-container in Angular

ng-container allows us to create a section in a template without introducing a new HTML element.

The ng-container does not render in the DOM, but content inside it is rendered.

ng-container is very useful Especially when working with structural directives like ngIf, ngFor.

we can conditionally append a group of elements in your application (by using ngif) or ng-container Can be used when we required Multiple Structural directives

ng-template in Angular

The ng-template is an Angular element, which contains the template.and The template does not render itself on DOM. These template elements only work in the presence of structural directives.


{{ngtemplate}}

There are few ways you can display the template.

1.Using the ngTemplateOutlet directive.

2.Using the TemplateRef & ViewContainerRef


ngTemplateOutlet in Angular

ngTemplateOutlet is a structural directive. We use it to insert a template (created by ngTemplate) in various sections of our DOM. we can pass data to the template and pass the template to child components.


{{ngtemplateoutlet}}
