Life Cycle Hooks in Angular

The hooks are executed in this order

constructor

It is invoked when Angular creates a component or directive by calling new on the class.

ngOnChanges

It is Invoked every time when there is a change in the input properties of the component.

ngOnChanges Example

ngOnChanges() will fire before ngOnInit() and every time parent Detais updated from its parent component.

The hook receives a SimpleChanges object that contains the previous and current values for the data-bound inputs properties.
